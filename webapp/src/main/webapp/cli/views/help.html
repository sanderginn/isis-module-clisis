Available commands:

  • menus
    Shows the available menus
  • menu <choice>
    Open the menu of choice
  • actions
    Show the available actions
  • action <choice>
    Invoke the action of choice
  • field <choice> <value>
    Fill out action parameter field of choice with value
  • submit
    Submit action parameters
  • get <choice>
    Get the object or collection of choice
  • properties
    Show the available properties
  • ollections
    Show the available collections
  • back
  • forward
  • help
    Shows this help menu
  • quit
    Exit the application
